Question(M2904/24). write a java code to collect ten(10) numbers from a user, then output all ten numbers given by the user, all even numbers and all odd numbers found in the numbers?

Solution:

1 import java.util.Scanner;
2 public class Main{
3 public static void main (String[]args){
4 Scanner sc = new Scanner(System.in);
5 int [] userInput = new int[10];

7 for(x=0; x<10; x++){

10 System.out.println("User Inputted values:");

13 }

18 }
19 }
20 System.out.println("Odd numbers discovered:");
21 for(x=0; x<10; x++){

25 }
26 }
27 }
